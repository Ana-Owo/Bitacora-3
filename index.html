Formularios CSS

Ideas principales

  • Se pide informacion al usuario
  • Se ejecuta con php
  • form: define un formulario
  • input: el mas importante y cambia su uso depen del atributo type
  • css nos proporciona forms para cambiar su estilo
  • text: para ingresar texto
  • submit: para enviar la infromación ingresada
  • color: seleccionar un color
  • password: ingresa contraseña y ocultarla
  • range: ingresa un rango
  • number: seleccionar un numero
  • checkbox: seleccionar varias opciones
  • radio: seleccionar una opción

Apuntes de la clase

Le pedimos la informacion al usuario y se captura la llenar los campos. Es parte de la web dinamica y tiene contacto con el sevidor. Se ejecuta con php.

Element form

El element html “form” Define un formulario que se utliza para recopilar informacion que el usario escribe

Input

Es el más importante en un formulario. Puede ser mostrado de diferentes fromas dependiendo del atributo “type”

Radio

sirve para seleccionar una de varias opciones

Checkbox

sirven par seleccionar varias opciones

File

sirve para cargar archivos a la base de datos

E-mail

sirve para ingresar un mail obligatoriamente

Color

sirve para seleccionar un color

Range

Sirve para seleccionar un rango

Password

sirve para ingresar una contraseña y esta se oculta

Submit

Sirve para enviar lo ingresado

date

sirve para mostrar un campo de fecha y se pueda seleccionar una

En CSS

CSS ofrece diversas propiedades para personalizar los campos de texto, desde los estilos básicos hasta los más avanzados

Ejemeplo:

input[type="text"] { border: 2px solid #3498db; }

Resumen

los formulario nos ayudan a que el usuario ingrese infromacion y se vaya a la base de datos, se utiliza con php.

La etiqueta forms es quien crea el formulario e input lo mas importante ya que se muestra de diferentes maneras dependiendo del atributo que se le asigne.

radio y checkbox sivern para seleccionar opciones

text y submit son para ingresar texto y enviar la información respectivamente.

los demás tipos sirven para seleccionar algun tipo de información con sus propios parametros.
