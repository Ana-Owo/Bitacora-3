Header

Ideas principales

  • El header es una de las partes más visibles de una página web y lo que los usuarios ven primero, por lo que debe ser ordenado, claro y fácil de entender.
  • Logotipo: Es esencial y refuerza la identidad de la marca. Generalmente, enlaza con la página de inicio para que los usuarios puedan volver a ella en cualquier momento.
  • Menú de Navegación (Nav): Permite a los usuarios navegar por las distintas secciones del sitio.
  • Buscador: Es importante para la usabilidad, ya que los usuarios nuevos pueden no estar familiarizados con el contenido del sitio.
  • Redes Sociales: Aunque son útiles para el mensaje de la marca, no deben ser el principal objetivo del header.
  • Elementos como el carrito, cuenta, pedidos, y listas de deseos son fundamentales en el header de una tienda online.
  • Botones como "Pedir presupuesto" o "Contactar" pueden destacarse en el header para enfatizar servicios o acciones específicas.
  • Barras Fijas: Útiles para mostrar anuncios, descuentos, o llamadas a la acción (CTAs). Sin embargo, se deben usar con moderación para no sobrecargar el espacio.

Apuntes de la clase

El header es una de las partes más importantes en una página web, pues es el elemento que los usuarios visualizan en primer lugar cuando llegan al proyecto. Por lo tanto, la cabecera de la web debe ser ordenada y limpia para que pueda ser visualizada con un solo vistazo y no dé lugar a confusión.

Debemos tener en cuenta que el header es un espacio genérico que tendrá el mismo diseño en todas las páginas de un sitio web. Lo primero que debes hacer es un listado con los elementos que vas a incluir en el header de tu web de forma obligatoria, ya sea por motivos de branding, navegación u otros.

Logotipos

En una web no puede faltar el logotipo, sirve como quia visual para el usuario y permite reforzar la imagen de la marca El logotipo en una web suele enlazar con la página de inicio. Es decir, permite que el usuario pueda volver al principio en todo momento. Se puede agregar al inicio o a la mitad.

Opcional

El menú o nav permite al usuario la navegacion entre las distintas secciones. Puede ser también agregado al header. Se puede colocar un solo menú o dos, siempre en función del proyecto y de su estructurea, si el proyecto tiene muchas secciones, posiblemente se necesite un menú doble, pero hay que evitarlo siempre que se pueda, pero si el proyecto tiene una gran estructura o muchas categorías, es ese caso se necesitara un doble menú.

Buscador

Otro elemento que puede estar en el header de una web es el buscador, es por cuestiones de usabilidad, ya que tu conoces muy bien tu web, pero para un usuario que entra por primeza vez es totalmente desconocida. Aquí hay varias opciones de diseño y ubicación: icono de lupa, barra buscadora, al lado del menú, debajo del logo, etc. Pero en la mayoría de los casos en el lado derecho de la web.

Redes sociales

Las redes sociales son un canal complementario para comunicar el mensaje de la marca o negocio, pero no el principal, que siempre debe ser la web. El objetivo de las redes sociales es amplifiar el mensaje para convertir en traficoc hacia la web, no al contrario.Es mas importante que un usuario realice una compra en la web que un like en instagram.

Si la web es una tienda online, se debe incluir en el header los elementos referentes al proceso de compra: carrito, cuenta, pedido, lista de deseos, inicio de sesión, etc.

Otro tipo de elemento que puedes incluir en la cabecera de tu web son botones destacados para enfatizar un servicio, llevar al usuario a una landing o animarle a que haga una accion especifica. Como ejemeplo un boton de “pedir presupuesto” o “pedir infromacion” como destacado.

Resumen

El header es una de las partes más importantes de una página web, ya que es lo primero que los usuarios ven al ingresar al sitio. Debe ser ordenado y claro para facilitar la navegación.

Logotipo: Refuerza la marca y generalmente enlaza con la página de inicio.

Menú de navegación: Facilita el acceso a las diferentes secciones del sitio, pudiendo ser simple o doble, según la cantidad de categorías.

Buscador: Importante para la usabilidad, especialmente para usuarios nuevos. Suele estar ubicado en el lado derecho.

Redes sociales: Complementan la comunicación de la marca, pero no deben ser el foco principal.

El header debe ser funcional y contener solo los elementos esenciales para que los usuarios puedan navegar de manera eficiente, realizar compras, y contactar con la empresa sin sobrecargar el diseño.
