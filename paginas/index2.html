Table

Ideas principales

  • Una tabla es un conjunto de datos organizados en filas y columnas, útil para mostrar información estructurada y facilitar la comparación entre valores.
  • table: Define la tabla.
  • thead: Agrupa las filas de encabezado.
  • th>: Celdas de encabezado (con texto en negrita y centrado por defecto).
  • tbody>: Agrupa las filas de datos (cuerpo de la tabla).
  • tr>: Define una fila dentro de la tabla.
  • td>: Define una celda de datos dentro de una fila.
  • caption>: Añade un título a la tabla.
  • tfoot>: Agrupa las filas del pie de la tabla (generalmente para sumarios).
  • rowspan: Especifica cuántas filas debe ocupar una celda.
  • colspan: Especifica cuántas columnas debe ocupar una celda.
  • Desde HTML 4, las tablas solo deben usarse para mostrar datos tabulados, no para el diseño de la página.

Apuntes de la clase

Las tablas en HTML nos permiten organizar los datos en filas y columnas. Una tabla en HTML consta de celdas de tabla dentro de filas y columnas.

¿Qué es una tabla?

Una tabla es un conjunto estructurado de datos distribuidos en filas y columnas (datos tabulados).

Una tabla permite buscar con rapidez y facilidad valores entre diferentes tipos de datos que indiquen algún tipo de conexión. Por ejemplo, una persona y su edad, o un día de la semana o el horario de una poiscina municipal.

Elementos de la tabla

thead>- encabezado de la tabla

th> - celda del encabezado de la tabla

tbody> - cuerpo de la tabla

tr> - renglon de la tabla

td> - celdas de datos

caption> - usados para insertar captions

tfoot> - footer de la tabla

Rowspan

Indica el número de filas que ocupara la celda. Por defecto ocupa una sola fila. td rowspam=3>. La celda ocupara el alto de 3 celdas normales.

Colspan

Indica el número de columnas que ocupara la celda. Por defecto ocupa una sola columna. td colspan=3>. Quiere decir que la celda actual se extiende en el ancho de dos celdas

Resumen

Las tablas en HTML sirven para organizar datos en filas y columnas. Originalmente, antes del uso de CSS, las tablas se utilizaban para el diseño y posicionamiento de páginas, pero este uso ha sido desaconsejado desde HTML 4.

Una tabla es un conjunto de datos organizados de manera estructurada, lo que permite comparar información de manera rápida y eficiente, como por ejemplo, mostrar la edad de una persona o los horarios de una piscina municipal.

Con los varios elementos de las tablas es posible organizar de manera facil los datos de relevamcia.
